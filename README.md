# 🍽️ Zomato Dataset Analysis

This project performs **data analysis and visualization on the Zomato restaurant dataset** to understand restaurant trends, ratings, locations, and cuisines. The goal of this project is to explore the dataset and extract useful insights using data analysis techniques.

---

## 📊 Project Overview

The **Zomato Dataset Analysis** project focuses on exploring restaurant data to identify patterns such as popular cuisines, restaurant ratings, location-based trends, and customer preferences. Data visualization techniques are used to present insights clearly.

---

## 🚀 Features

* 📂 Data cleaning and preprocessing
* 📊 Exploratory Data Analysis (EDA)
* 📈 Data visualization using graphs and charts
* ⭐ Analysis of restaurant ratings
* 🌍 Location-based restaurant analysis
* 🍛 Cuisine popularity analysis

---

## 🛠️ Technologies Used

* **Python**
* **Pandas** – Data manipulation and analysis
* **NumPy** – Numerical computations
* **Matplotlib** – Data visualization
* **Seaborn** – Advanced data visualization
* **Jupyter Notebook** – Project development and analysis

---

## 📂 Dataset

The dataset contains information about restaurants such as:

* Restaurant Name
* Location
* Cuisines
* Average Cost
* Rating
* Votes

This data helps analyze restaurant trends and customer preferences.

---

## 📊 Visualizations Included

The project includes several visualizations such as:

* 📊 Bar plots of restaurant ratings
* 📈 Scatter plots showing relationship between variables
* 📦 Box plots for rating distribution
* 🔥 Heatmaps for correlation analysis
* 📍 Location-based restaurant count analysis
